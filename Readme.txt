Desenvolvimento.

A aplicação foi desenvolvida em PHP, com o sistema Xampp.

Para iniciar:

1) Colocar os arquivos em uma pasta de localhost do apache;
2) Criar dois bancos de dados no PhPAdmin:

samuca
login

Obs: suário e senha padão do PhPAdmin: root e ""

3) Após criado os bancos de dados, criar duas tabelas;

- Para criar as tabelas, basta acessar (pela URL) os arquivos pdo_tabela.php e usuarios_tabela.php; ambos criam as tabelas.

4) O sistema estará pronto para ser usado;

Pasta inicial: desafio_php
